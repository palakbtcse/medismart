    Process
    How MediSmart Works

    A six-stage pipeline from raw patient input to a severity-weighted, dataset-driven recommendation.

    1
    Patient Input
    User provides age, gender, and selects currently experienced symptoms from 86 real clinical symptoms derived from the dataset.
    2
    Data Preprocessing
    Categorical variables are encoded (Male=0, Female=1), symptoms converted to binary feature vectors, and data normalised for model input.
    3
    Severity-Weighted Matching
    Each disease is scored by how many selected symptoms it matches, weighted by clinical severity scores from the Symptom-severity dataset.
    4
    ML Classification
    Decision Tree and Random Forest classifiers analyse the feature vector, identify symptom patterns, and predict the most appropriate disease class.
    5
    Full Recommendation Output
    Top 3 matched conditions are displayed with disease description, symptom match percentage, severity badge, medications, and step-by-step precautions.
    6
    Admin Dataset View
    Administrators can browse all 41 diseases with their symptoms and medications from the integrated dataset via the Admin dashboard.
